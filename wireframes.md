# Wireframes (text descriptions)

1) 홈 페이지 — 포스트 카드 목록 + 검색바

- 상단: `Header` (로고, 검색 입력, 로그인 버튼)
- 메인: 가로 그리드(모바일 1열, md 이상 2열)
  - 좌측: 포스트 카드 목록
    - 각 카드: `Card` 안에 `CardTitle`(링크), `CardDescription`(요약), 메타(작성자, 날짜)
  - 우측(데스크톱): 사이드바(프로필 요약, 인기 글, 태그 목록)
- 하단: `Footer`

스타일: 간결/화이트 스페이스 중심, `Card`에 라운드와 섀도우, 텍스트는 `--foreground` 사용

2) 포스트 작성 페이지 — 제목, 내용 입력 폼 + 제출 버튼

- 레이아웃: 중앙 max-width 컨테이너
- 폼 필드:
  - 제목: `Input` (large)
  - 본문: `textarea` 또는 리치에디터 컴포넌트
  - 태그 입력: 작은 `Input` + 태그 토큰
  - 버튼: `Button`(Primary) - `저장`, `임시저장`, `미리보기`

스타일: 입력 중심, 간결한 액션 영역, 모바일 우선 디자인

Notes:
- 추천: v0.dev 또는 Copilot Vision으로 시각 와이어프레임을 생성하면 검증에 용이합니다.
